# BOJ 1002, 터렛

- RANK : Silver
- Language : Python
- [터렛](https://www.acmicpc.net/problem/1002)

<br/>

# 문제 설명

- 레이더 두개를 사용해서 상대방의 위치를 측정할때 위치로 가능한 좌표를 출력해라.
- 원의 교점의 개수에 관한 수학 문제

<br/>

# 두 원의 교점

- R-r, d, R+r 값을 비교하여 외부, 외접, 두 점, 내접, 내부 로 case를 나눈다.
- 두 원의 중심이 같을 때의 예외처리를 해야한다.
