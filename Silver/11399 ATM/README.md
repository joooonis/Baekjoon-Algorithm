# BOJ 11399, ATM

- RANK : Silver
- Language : Python

# 문제 설명

- ATM이 1대 밖에 없다. N명의 사람이 줄을 서고 있다. 각각 돈을 인출하는데 걸리는 시간이 주어질때 각 사람이 ATM을 이용하는데 걸리는 시간의 합의 최솟값을 구하시오.
  <br/>

### Sorting(정렬)

- 걸리는 시간이 적은 순으로 인출을 할 때가 최소이므로 정렬해서 합을 구한다.
