# 문제

- 두 집합이 주어졌을 때 대칭차집합의 원소의 개수를 구하여라

  <br />

# set in python

- 생성자 : set()
- 교집합 : A & B, A.intersection(B)
- 합집합 : A | B, A.union(B)
- 차집합 : A - B, A.difference(B)
- 대칭차집합 : A ^ B
- 추가 : A.update([1,2,3])
- 제거 : A.remove(1)
